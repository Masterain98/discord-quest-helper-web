🚀 快速開始

只需 4 个简单步骤即可開始

1

下載并執行

下載最新版本，解壓縮 .zip 文件，然后執行 Discord Quest Helper.exe。

2

登录 Discord

点击 Auto Detect Token 自动登录，或使用 Manual Input 手動输入。自动偵測需要 Discord 客户端在后台執行。

3

完成任务

瀏覽您的任务并点击開始。影片任务直接在應用内完成。遊戲任务需要 Discord 客户端在線。

4

領取奖励

完成任务后，通过 Discord 客户端手動領取奖励。此應用自动化進度但不会領取奖励。

⚠️
重要提示

此工具仅供学习目的。使用此工具可能違反 Discord 的服務條款。作者不对任何後果负责。使用風險自负。

📖 使用指南

了解如何完成不同类型的任务

📺

影片与直播任务

  1. 在首页找到影片或直播任务
  2. 点击 "🎬 Start Watching" 或 "📡 Start Streaming" 按鈕
  3. 應用自动向 Discord 服务器提交進度
  4. 等待任务完成
💡 進度完全由此應用處理。无需观看任何内容或保持 Discord 在線。可在設定中调整完成速度 (1-10x)。
🎮

遊戲任务

  1. 在首页找到遊戲任务
  2. 直接在任务卡片上点击 "🎮 Start Playing" 按鈕
  3. 應用自动建立并執行模擬遊戲进程
  4. Discord 客户端偵測到进程并更新您的進度
💡 无需下載或安裝實際遊戲。Discord 客户端必须在后台執行。

🕹️ 遊戲模擬器

模擬任何 Discord 支持的遊戲以显示 Rich Presence

⚙️

自定义遊戲模擬

遊戲模擬器允许您为 Discord 支持的任何遊戲建立和執行模擬进程。这对以下情况很有用：

  • 完成特定遊戲的任务
  • 在 Discord 上显示自定义 Rich Presence 狀態
  • 测试 Discord RPC 集成
🛠️

使用方法

  1. 进入 Game Simulator 標籤頁
  2. 从 Discord 遊戲資料庫中搜索并選擇一个遊戲
  3. 選擇可執行文件名和安裝路径
  4. 点击 "Create Game" 生成模擬可執行文件
  5. 点击 "Run" 啟動进程并连接 Discord RPC
  6. 完成后点击 "Stop"

常见问题

常见问题解答

Discord Token 是 Discord 客户端本地存储的身份驗證凭证，用于维护您的登录会话。此應用程序直接从 Discord 的本地存储读取 Token，无需通过瀏覽器开发者工具或控制台脚本手動提取。

Token 仅在應用程序会话期间存储在内存中，从不持久化到磁盘。所有 API 请求仅通过 HTTPS 傳輸到 Discord 的官方服务器。應用程序是开源的，允许完整的代码审查和驗證。

不需要。遊戲模擬器建立一个轻量级进程，Discord 的 Rich Presence 系统将其识别为目标遊戲。这消除了下載、安裝或拥有實際遊戲的需要，节省了磁盘空间和带宽。

影片任务使用 Discord 的 HTTP API，允许此應用程序直接向 Discord 服务器提交观看進度。遊戲任务依赖 Discord 的 Rich Presence 协议，需要 Discord 桌面客户端偵測正在執行的遊戲进程并向服务器报告活动。

高级篩選功能可以按奖励类型（裝飾品、物品等）、任务类型（影片、遊戲、直播）和完成狀態篩選任务。此功能帮助用户快速找到符合其条件的特定任务。

應用程序支持 Discord Stable、Discord Canary 和 Discord PTB（公开测试版）。自动偵測适用于您系统上安裝并執行的任何版本。

目前，應用程序仅支持 Windows。Token 提取依赖 Windows Data Protection API (DPAPI) 进行解密。跨平台支持可能会在未来版本中考虑。

要完全卸载應用程序，请刪除應用程序資料夾。快取文件（模擬遊戲可執行文件）存储在 Documents\DiscordQuestGames。刪除此資料夾以清除所有快取数据。您可以直接从應用内的設定页面访问此資料夾。
