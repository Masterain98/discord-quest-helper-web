  4. Discord client detects the process and updates your progress

Custom Game Simulation

The Game Simulator allows you to create and run simulated processes for any game that Discord supports. This is useful for:

  • Completing game quests for specific titles
  • Displaying custom Rich Presence status on Discord
  • Testing Discord RPC integrations
üõ†Ô∏è

How to Use

  1. Go to the Game Simulator tab
  2. Search and select a game from Discord's game database
  3. Choose an executable name and installation path
  4. Click "Create Game" to generate the simulated executable
  5. Click "Run" to start the process and connect Discord RPC
  6. Click "Stop" when finished

Frequently asked questions

A Discord token is an authentication credential stored locally by the Discord client to maintain your login session. This application reads the token directly from Discord's local storage without requiring manual extraction through browser developer tools or console scripts.

The token is stored in memory only during the application session and is never persisted to disk. All API requests are transmitted over HTTPS exclusively to Discord's official servers. The application is open source, allowing full code review and verification.

No. The Game Simulator creates a lightweight process that Discord's Rich Presence system recognizes as the target game. This eliminates the need to download, install, or own the actual game, saving both disk space and bandwidth.

Video quests utilize Discord's HTTP API, allowing this application to submit watch progress directly to Discord servers. Game quests rely on Discord's Rich Presence protocol, which requires the Discord desktop client to detect running game processes and report activity to the server.

Advanced Filter enables filtering quests by reward type (cosmetics, items, etc.), quest type (video, game, stream), and completion status. This functionality helps users quickly locate specific quests matching their criteria.

The application supports Discord Stable, Discord Canary, and Discord PTB (Public Test Build). Auto-detection works with whichever version is installed and running on your system.

Currently, the application is Windows-only. Token extraction relies on Windows Data Protection API (DPAPI) for decryption. Cross-platform support may be considered in future releases.

To completely remove the application, delete the application folder. Cache files (simulated game executables) are stored in Documents\DiscordQuestGames. Delete this folder to remove all cached data. You can access this folder directly from the Settings page within the app.
